Backup Information
What's included in backups:
  • All competition data
  • Athlete registrations
  • Attempt records and scores
  • Timer and event configurations
  • System settings
Backup recommendations:
  • Create backups before major events
  • Keep at least 3 recent backups
  • Store important backups offline
  • Test backup restoration periodically
Important Notes
Restore Warning

Backup restoration is currently disabled for safety. Restoring a backup will completely replace the current database and cannot be undone.

Backup Storage

Backups are stored in the application's backup directory. For production use, consider storing backups in a separate, secure location.

Data Safety

The application automatically prevents data corruption by validating backup files before any operations.
